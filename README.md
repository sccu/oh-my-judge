# Oh-My-Judge (OMJ)

**Oh-My-Judge**는 사람이 수행하는 온라인 서비스 평가 작업을 LLM 에이전트로 자동화하는 프로젝트입니다.

## 문서 가이드

프로젝트의 모든 주요 문서와 설계 내용은 `/docs` 디렉토리 아래에 관리되고 있습니다.

- **[README (Docs)](./docs/README.md)**: 프로젝트 개요 및 핵심 목표
- **[DESIGN](./docs/DESIGN.md)**: 시스템 설계 및 스텔스 전략
- **[SPECIFICATION](./docs/SPECIFICATION.md)**: 태스크 정의 및 평가 규격
- **[RESEARCH_LOG](./docs/RESEARCH_LOG.md)**: 기술 조사 및 실험 기록
